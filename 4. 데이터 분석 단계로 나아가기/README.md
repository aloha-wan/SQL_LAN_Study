### 1️⃣ COUNT(), MAX(), MIN(), AVG()

COUNT() 안에 컬럼을 넣는다면 NULL 을 COUNT 하지 않기 때문에 유의해야한다.

NULL과 상관없이 COUNT를 하기 위해선 따로 컬럼을 넣지 않고 COUNT(*)로 작성 하면 된다.



**MAX()** 모든 ROW 중 **최대값**, **MIN()** 모든 ROW 중 **최소값**, **AVG()** 안에 컬럼을 넣어 **평균값**을 나타낸다.
